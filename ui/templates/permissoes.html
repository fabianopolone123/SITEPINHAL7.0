Como funciona

1) Configure os menus por grupo.

2) Vincule os usuarios aos grupos.

3) Use excecao por usuario so quando precisar.

Regra de prioridade: se um usuario tiver excecao individual marcada, ela prevalece sobre o grupo e a linha de grupos fica bloqueada para evitar conflito.
